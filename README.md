# Task Manager API 🚀

A simple backend application built using Spring Boot and MongoDB.

## Features

* Create Task
* Get All Tasks
* Get Task by ID
* Update Task
* Delete Task
* Validation & Error Handling

## Tech Stack

* Java
* Spring Boot
* MongoDB
* Maven

## API Endpoints

### Create Task

POST /tasks

### Get All Tasks

GET /tasks

### Get Task by ID

GET /tasks/{id}

### Update Task

PUT /tasks/{id}

### Delete Task

DELETE /tasks/{id}

## Run Locally

```bash
./mvnw spring-boot:run
```
